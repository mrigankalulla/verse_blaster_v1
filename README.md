VerseBlaster ✦

Play it live: https://mrigankalulla.github.io/verse_blaster_v1/VerseBlaster.html

A browser-based poetry game inspired by Emily Blaster — the game built by Sadie Green in Gabrielle Zevin's Tomorrow, and Tomorrow, and Tomorrow. In the novel, Sadie creates a shooter where words from Emily Dickinson poems fall from the sky. This is my remix.


What it does
Before each round, a poem is read aloud — Mary Oliver, Rumi, Tagore, Byron, Agha Shahid Ali, Langston Hughes, and others. Then the words from that poem fall down the screen. Shoot them before they escape.

Words explode into flowers and hearts. Soft ambient jazz plays in the background. The whole thing is designed to feel like stepping into a candlelit bookshop.


Why I built it
I've always been drawn to the intersection of culture and product — how a well-designed experience can make something feel like it belongs in your life. VerseBlaster started as a creative experiment: could a word game feel literary rather than gamey? Calm rather than frantic?

It also gave me a chance to go deeper into how products are actually built — from canvas rendering and collision detection to speech synthesis and procedural audio generation.


Built with
HTML5 Canvas — all game rendering, particles, animations
Web Speech API — reads each poem aloud before play begins
Web Audio API — procedurally generated ambient jazz (walking bass, chord pads, brushed hi-hats)
Google Fonts — Playfair Display for that editorial serif feel
Zero frameworks. One file.
